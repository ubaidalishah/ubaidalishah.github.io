### 📌 University Application Tracker

A simple **University Application Tracker** that lets you **add, manage, and download** your university application list in **PDF format**.

---

## 🚀 Features

✅ Add university details (Name, Website, Country, Deadline, Applied Status)  
✅ **Responsive design** – Works on mobile & desktop  
✅ **Dynamic PDF generation** – Download your application list  
✅ **Toggle applied status** (✅/❌)

## 🛠 Setup Instructions

1. **Clone the repo**

   ```bash
   git clone https://github.com/ubaidalishah/ubaidalishah.github.io.git
   cd University-Application-Tracker
   ```

2. **Open `index.html` in a browser**  
   You can simply **double-click** the `index.html` file or run a local server:
   ```bash
   python3 -m http.server 8000
   ```
   Then open **http://localhost:8000/** in your browser.

---

## 📚 Usage

1. **Enter university details** in the input fields.
2. **Click "Add"** to store the entry in the table.
3. **Toggle applied status** (✅/❌) by clicking the icon.
4. **Click "Download PDF"** to save your list (visible only when an entry exists).
